# HNS CLI

REST/CLI and RPC clients for handshake.

## Install

```
sudo add-apt-repository ppa:pcfreak30/hns-cli
sudo apt-get update
sudo apt-get install hns-cli
```

## Usage

Programs available:

* hsd-cli
* hsd-rpc
* hsw-cli
* hsw-rpc

CLI/API Docs: https://hsd-dev.org/api-docs/
